## Commands

```bash
# make build-dev
$ build application for development

# make start
$ start application in development mode

# make build-prod
$ build application for production

# make start-prod
$ start application in production mode

# make stop
$ stop application

# make logs
$ open application logs

# make ssh
$ go inside application container

# make info
$ get container info

# make info
$ get container ip

# make list
$ get containers list
```
